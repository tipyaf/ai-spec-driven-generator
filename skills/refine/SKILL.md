name: refine
description: Refine a user story — break it down into actionable tasks with acceptance criteria and acceptance tests. Use when starting work on a new feature or story.
---

Load and follow the instructions in the refinement agent:
@../agents/refinement.md

Then load the Product Owner agent for acceptance tests:
@../agents/product-owner.md

## Workflow
1. Read the story/feature description from the user
2. Follow the refinement agent workflow to break it down
3. Ensure every sub-task has acceptance criteria AND machine-verifiable acceptance tests
4. Present the breakdown to the user for validation
