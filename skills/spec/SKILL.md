@../agents/architect.md

## Workflow
1. PO: Gather requirements, challenge assumptions, write spec
2. UX/UI: Design information architecture, flows, components (if UI project)
3. Architect: Plan architecture, select stack, create implementation manifest
4. Present complete spec to user for validation
